# 多快捷键点击

一个 Tampermonkey 用户脚本，用于在指定页面中配置快捷键任务。当前支持 XPath 点击、分组快捷粘贴、执行自定义 JavaScript，并提供可视化配置面板。

## 功能

- 点击任务：为一个任务配置一个快捷键，可按顺序点击多个 XPath。
- 快捷粘贴：使用统一快捷键打开文本选择列表，在鼠标位置选择要粘贴的文本。
- 粘贴分组：可创建多个文本分组，并在配置面板选择当前使用的分组。
- JS 任务：配置快捷键执行自定义 JavaScript，支持 `await delay(ms)` 和 `toast(msg)`。
- 配置管理：支持导入、导出、保存和恢复默认配置。
- 兼容旧配置：旧版一条粘贴文本一个快捷键的配置会自动迁移到默认分组。

## 安装

1. 安装浏览器扩展 Tampermonkey。
2. 新建用户脚本。
3. 将 `index.js` 的内容复制到脚本编辑器中并保存。
4. 打开匹配页面后，右侧会出现 `TM` 配置按钮。

默认匹配页面：

```js
// @match        https://aidp.juejin.cn/*
// @match        file:///*
```

如需在其他页面使用，请在 `index.js` 顶部添加对应的 `@match` 规则。

## 使用

点击页面右侧的 `TM` 按钮打开配置面板。

### 点击任务

1. 点击 `新增点击任务`。
2. 输入任务名称。
3. 点击快捷键输入框，然后直接按下要绑定的快捷键。
4. 在 XPath 列表中填写要点击的元素 XPath，一行一个。
5. 可设置点击间隔，任务执行时会按顺序点击。

### 快捷粘贴

1. 在 `快捷粘贴` 区域设置统一快捷键，默认是 `Ctrl+Shift+V`。
2. 选择或新增一个粘贴分组。
3. 点击 `新增文本`，填写文本名称和内容。
4. 在页面输入框中聚焦后，按统一快捷键。
5. 脚本会在鼠标位置弹出当前分组的文本列表，点击后自动粘贴。

### JS 任务

1. 点击 `新增 JS 任务`。
2. 输入任务名称和快捷键。
3. 填写要执行的 JavaScript 代码。
4. 可在代码中使用：

```js
await delay(500);
toast('执行完成');
```

## 配置导入导出

- 导出配置：点击 `导出配置`，会下载当前完整配置 JSON 文件。
- 导入配置：点击 `导入配置`，选择 JSON 文件后会覆盖当前配置。
- 保存配置：点击 `保存全部配置` 会将当前面板内容保存到 Tampermonkey 存储中。
- 恢复默认：点击 `恢复默认` 会清空当前配置并恢复脚本内置默认值。

## 注意事项

- 快捷键在配置面板内不会触发任务，避免编辑配置时误执行。
- 粘贴功能会优先写入剪贴板，再尝试插入到当前聚焦的输入框或可编辑区域。
- 如果当前焦点不可粘贴，文本仍会被复制到剪贴板。
- XPath 点击任务依赖页面结构，页面更新后可能需要重新调整 XPath。
